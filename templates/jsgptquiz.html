JavaScript Quiz

Welcome to the JavaScript quiz. This quiz will test your knowledge on various fundamental aspects of JavaScript. Here are the key topics you should understand:

  • Data Types: JavaScript has several data types, including:
    • String: Represents text, e.g., "Hello, world!".
    • Number: Represents both integer and floating-point numbers, e.g., 42 or 3.14.
    • Boolean: Represents true or false.
    • Object: A collection of properties, e.g., { name: "John", age: 30 }.
    • Undefined: A variable that has been declared but not assigned a value.
    • Null: Represents the intentional absence of any object value.
  • Functions: Functions are blocks of code designed to perform a particular task. You can declare functions in several ways:
    • Function Declaration: function myFunction() { /* code */ }
    • Function Expression: var myFunction = function() { /* code */ };
    • Arrow Function: let myFunction = () => { /* code */ };
  • Operators: JavaScript has various operators to compare values:
    • ==: Compares values for equality after converting both to a common type.
    • ===: Strict equality operator that compares both value and type.
  • Arrays: Arrays are used to store multiple values in a single variable. They can hold different data types and are zero-indexed. You can manipulate arrays using methods such as:
    • push(): Adds an element to the end of an array.
    • pop(): Removes the last element from an array.
    • shift(): Removes the first element from an array.
    • unshift(): Adds an element to the beginning of an array.
  • Falsy Values: In JavaScript, some values are considered false when evaluated in a boolean context. These include:
    • 0
    • "" (empty string)
    • null
    • undefined
    • NaN (Not-a-Number)
    • false
  • ES6 Features: ECMAScript 6 (ES6) introduced several new features, including:
    • let and const: Block-scoped variables.
    • Arrow Functions: Shorter syntax for function expressions.
    • Template Literals: Allows embedding expressions inside string literals using backticks (`) and ${}.
    • Destructuring: A syntax for extracting values from arrays and objects.
