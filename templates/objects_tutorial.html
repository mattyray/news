Objects in JavaScript

This tutorial introduces JavaScript objects, explains how they work, and provides examples for beginners.

What is an Object?

In JavaScript, an object is a complex data structure that stores collections of key-value pairs called properties. The key is a string (or property name), and the value can be any data type, such as numbers, strings, arrays, or other objects.

Objects are flexible and used extensively in JavaScript to organize and manipulate data.

Creating an Object

You can create an object in JavaScript using object literal syntax:


let person = {
    name: "John",
    age: 30,
    isStudent: false
};

The object person has three properties: name, age, and isStudent. You can access these properties using dot notation or bracket notation:


console.log(person.name);  // Output: John
console.log(person['age']);  // Output: 30

Adding and Modifying Properties

You can add new properties or modify existing ones:


person.city = "New York";  // Adds a new property 'city'
person.age = 31;  // Modifies the 'age' property

Methods in Objects

A method is a function that is a property of an object. Methods allow you to add behaviors to objects:


let person = {
    name: "John",
    greet: function() {
        console.log("Hello, my name is " + this.name);
    }
};

person.greet();  // Output: Hello, my name is John

The keyword this refers to the current object, enabling access to other properties or methods within the object.

Nested Objects

Objects can also contain other objects, creating a structure known as nested objects:


let company = {
    name: "Tech Corp",
    address: {
        street: "123 Main St",
        city: "San Francisco",
        state: "CA"
    }
};

console.log(company.address.city);  // Output: San Francisco

Looping Through Object Properties

You can use a for...in loop to iterate over an object's properties:


let person = {
    name: "John",
    age: 30,
    isStudent: false
};

for (let key in person) {
    console.log(key + ": " + person[key]);
}

Object Methods

JavaScript provides several built-in methods for working with objects:

  • Object.keys(): Returns an array of an object's property names.
  • Object.values(): Returns an array of an object's property values.
  • Object.entries(): Returns an array of key-value pairs.

let car = {
    make: "Tesla",
    model: "Model 3",
    year: 2021
};

console.log(Object.keys(car));  // Output: ['make', 'model', 'year']
console.log(Object.values(car));  // Output: ['Tesla', 'Model 3', 2021]
console.log(Object.entries(car));  // Output: [['make', 'Tesla'], ['model', 'Model 3'], ['year', 2021]]
